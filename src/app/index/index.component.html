Setup Guide

1.Download the modded opengoal (Code -> Download ZIP)

2.Unzip and open the folder inside.

3.Drag and drop your jak 1 iso file onto the extractor and let it run until the game starts by itself, close the game afterwards.

4.Download the latest release (installer or portable) above.

5.Install/Run the client.

6.Point it to the gk.exe file inside the opengoal folder that was unzipped on step 2 when prompted to.

7.Read the "Help/Info" page found on the lobby page.

8.Create or join a lobby/run and play.
